# what is this project?

text editor from scratch in c

credit: https://github.com/antirez/kilo/blob/master/kilo.c#L436


# how to compile and run?


1. cc kilo.c
2. ./a. out


# how to save and exit?

1. ctrl + s to save
2. ctrl + q to exit


# feature

1. write
2. enter to swap row
3. delete text
4. save file (ctrl + s)
5. quit file (ctrl + q)
6. find (ctrl + f)
